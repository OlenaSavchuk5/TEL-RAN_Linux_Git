Укажите следующие команды:

1. git reset - для полной очистки индекса

2. git diff - для просмотра разницы между рабочим каталогом и последним KOMMUTOM 

3. git checkout -b fixbug - для создания новой ветки и одновременного переключения на нее

4. git diff ..bugfix - для сравнения веток между собой git diff master bugfix 

5. git push -u origin develop - для загрузки ветки develop Ha удаленный репозиторий (GITHUB)

6. git branch -D fixbug - для удаления локальной ветки 

7. Необходимо находиться на основной ветки master и затем git merge fixbag - для слияния веток 

8. git show 24ba:index.html - для открытия (просмотра) файла по состоянию Ha какой-то коммит 
24ba - первые четыре значения от длинного номера коммита
